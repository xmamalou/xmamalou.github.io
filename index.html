Science-Tex

Science-Tex is a LaTeX template for writing beautiful and professional scientific papers, without much hassle. Comes in double column and single column variants, and is easily customizable! Check it out by clicking the image on the right!

Termite

Termite is a library for memory and thread management. It's made to be simple to use and also integrates Vulkan for GPU memory management and usage. It's still in development, but you can check it out by clicking the image on the right!

Dragonfly

Dragonfly is a rendering and physics engine, designed to be small and simple to use. It's written in C but allows easy integration with other languages. It uses Vulkan and GLFW. It's still in development, but you can check it out by clicking the image on the right!

Linear Algebra Fundamentals (Incomplete)
